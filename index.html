09 / Best Practices

Do's and Don'ts

✅ Best Practices

  • Separate LLM and GPU rendering (different containers)
  • Use async queue (webhook, don't block)
  • Retry with exponential backoff for all APIs
  • Cache intermediate results (TTS, b-roll)
  • Version content (script v1 → audio v1 → video v1)
  • Health checks: GPU usage, disk, queue depth
  • Graceful degradation (HeyGen down → EchoMimicV3)

❌ Anti-Patterns

  • Don't store videos on GPU VPS long-term (expensive)
  • Don't use sync calls for video generation (minutes!)
  • Don't depend on single provider (HeyGen can go down)
  • Don't pass credentials over unencrypted channels
  • Don't run Ollama + EchoMimicV3 simultaneously on 24GB
  • Don't use Telegram for file transfer (compression)
  • Don't store videos in Git (large binaries)
